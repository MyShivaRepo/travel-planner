# Process
Ce fichier décrit le processus suivi par l'utilisateur pour plannifier son voyage.   

L'utilisateur saisi sa destination.   
(Ce peut être un pays, une région ou bien une ville)   

Le système détermine alors les sites "à ne pas manquer" pour cette destination.   

Le système présentes sites dans un tableau :
- Rang du site
- Nom du site   
- Description   
- Type de site (Nature, Histoire, Architecture, ...)   
- Latitude   
- Longitude
Ce tableau peut-être trié selon chaque colonne en cliquant sur l'en-tête de colonne

Le système positionne également ces sites sur une carte géographique dont l'échelle est adaptée pour visualiser l'ensemble du voyage.

Si l'utilisateur valide la liste des sites, alors le système génére une proposition de voyage jour par jour
Pour chaque jour, le système propose quel(s) site(s) visiter
Dans quel hotel séjourner le soir
Et dans quel restaurant (bien noté) se restaurer le soir
